### Markdown是什么
 百度百科解释是，Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。 👉  [官网](http://www.markdown.cn/)
***
### 为什么使用Markdown
 能用简单的标记语言实现文字排版，且现在很多平台都支持，用起来就对啦！
***
### Markdown基础用法
#### 标题
 1、说明：标题标记有两种方式，一种是通过`=`和`-`符号标记，另一种是通过用`#`号可表示 1-6 级标题，一级标题对应一个`#`号，二级标题对应两个`#`号，以此类推(此方法较常使用)
 2、写法👇
 ```
 一级标题
 ===
 二级标题
 ---
 ```
 ```
 # 一级标题
 ## 二级标题
 ### 三级标题
 #### 四级标题
 ##### 五级标题
 ###### 六级标题
 ```
 3、效果👇
 
 ![字体](https://img-blog.csdnimg.cn/20191207170233551.png)
 #### 段落格式
  ##### 【1】字体
  1、说明：`*`为斜体，`#`为粗体
  2、写法👇
  ```
  *斜体*
  _斜体_
  **粗体**
  __粗体__
  ***粗斜体***
  ___粗斜体___
  
  ```
  3、效果👇
  *斜体*
  _斜体_
  **粗体**
  __粗体__
  ***粗斜体***
  ___粗斜体___
  
  ##### 【2】分割线
  1、说明：用三个以上的`*`、`-`即可建立一个分隔线，行内不能有其他东西。也可以在`*`或是`-`中间插入空格。
  2、写法👇
  ```
  ***
  ---  
  ```
  3、效果👇
  ***
  ---
  
  ##### 【3】删除线
  1、说明：在文字的两端加上两个波浪线`~~`即可
  2、写法👇
  ```
  ~~删除线~~
  ```
  3、效果👇
  ~~删除线~~
  
  ##### 【4】下划线
  1、说明：在文字的两端加上两个波浪线`~~`即可
  2、写法👇
  ```
  ~~删除线~~
  ```
  3、效果👇
  ~~删除线~~
  
 #### 列表
  1、说明：支持有序列表和无序列表。
  无序列表使用`*`、`+`或是`-`作为列表标记；
  有序列表使用数字并加上`.`号来表示，如果前面标记的数字不对会自动生成对应的准确数字哦
  2、写法👇
  ```
  * 第一项
  * 第二项
  * 第三项
  
  + 第一项
  + 第二项
  + 第三项
  
  
  - 第一项
  - 第二项
  - 第三项
    
  1. 第一项
  2. 第二项
  3. 第三项
  ```
  3、效果👇
  * 第一项
  * 第二项
  * 第三项
  
  + 第一项
  + 第二项
  + 第三项
  
  
  - 第一项
  - 第二项
  - 第三项
  
  1. 第一项
  2. 第二项
  3. 第三项
    
  
 #### 区块
  1、说明：在段落开头使用`>`符号 ，然后后面紧跟一个空格符号
  2、写法👇
  ```
  > 区块引用
  ```
  3、效果👇
  > 区块引用
   
 #### 代码
  1、说明：单个或片段的代码可以用反引号`包裹，代码区块用```包裹
  2、写法👇
  ````
  `片段`
  ```
  代码区块
  ```
  ````
  
  3、效果👇
  
   `片段`
   ```
   代码区块
   ```
    
 #### 链接
  1、说明：使用`[链接名称](链接地址)`和`<链接地址>`两种写法
  2、写法👇
  ````
  [Linda's Blog](https://www.linda1994.top/)
  <https://www.linda1994.top/>
  ````
  
  3、效果👇
  
  [Linda's Blog](https://www.linda1994.top/)
  <https://www.linda1994.top/>
    
 #### 图片
  1、说明：`![alt 属性文本](图片地址)`和`![alt 属性文本](图片地址 "可选标题")`两种写法
  2、写法👇
  ````
  ![Linda's Alipay，扫码来赞赏吧](https://img-blog.csdnimg.cn/20191208202108184.jpeg)
  
  ````
  
  3、效果👇
  ![Linda's Alipay，扫码来赞赏吧](https://img-blog.csdnimg.cn/20191208202108184.jpeg)
    
 #### 表格
  1、说明：使用 `|` 来分隔不同的单元格，使用 `-` 来分隔表头和其他行。
  `-:` 设置内容和标题栏居右对齐。
  `:-`设置内容和标题栏居左对齐。
  `:-:` 设置内容和标题栏居中对齐。
  2、写法👇
  ````
  [Linda's Blog](https://www.linda1994.top/)
  <https://www.linda1994.top/>
  ````
  
  3、效果👇
  
 | 左对齐 | 右对齐 | 居中对齐 |
 | :-----| -----: | :-----: |
 | 单元格 | 单元格 | 单元格 |
 | 单元格 | 单元格 | 单元格 |
  
  ⚠️markdown是在td上添加了属性`align="xx"`，因为小举我这个页面td的css已经设置成了`text-align: left;`，样式已被覆盖，所以页面都会展示成左对齐哦～
